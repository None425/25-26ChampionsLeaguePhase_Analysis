# 챔스 리그페이즈 지표 분석
분석 기간 : 2026-02-02 ~ 2026-02-05
## 프로젝트 개요
본 프로젝트는 2024-25 시즌 개편된 UEFA 챔피언스리그 리그 페이즈 데이터를 활용하여 팀별 경기력을 정량적으로 분석하고,    
이를 바탕으로 플레이오프 대진 결과를 예측하는 것을 목표로 한다.
리그 페이즈에서는 총 36개 팀이 팀당 8경기를 치르며, 홈 4경기·원정 4경기로 구성된다.   
본 분석에서는 해당 경기 데이터를 기반으로 팀별 평균 지표를 산출하고,  
홈 어드밴티지의 존재 여부를 검증한 뒤 이를 반영한 확률 기반 예측 모델을 구축하였다.

## 활용 데이터 
1. 25-26챔피언스리그 리그페이즈 스탯 - https://www.kaggle.com/datasets/johntocci/champions-league-matches-2025-2026
- 분석 대상 : 리그 페이즈 전체 36개 팀
- 주요 변수: goals(득점), possession(점유율), shots_on_target_pct(유효슈팅률), saves_pct(선방률)
